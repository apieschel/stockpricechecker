Nasdaq Stock Price Checker

  1. Using the form below, you can GET /api/stock-prices with form data containing a Nasdaq stock ticker.
  2. The returned stock object will contain the stock name (a string, also known as the "ticker"), the price (a decimal in string format), and the number of likes (an integer).
  3. If the like field is passed as true(boolean) via the checkbox, then the user's like will be added to the stock object. Only one like is permitted per IP address.
  4. If two stocks are requested, the returned object will be an array with both stocks' info, but instead of likes, they will contain rel_likes (the difference between the likes on both).
  5. The Alpha Vantage stock API is used for this project: https://www.alphavantage.co/.

Example usage:

/api/stock-prices?stock=goog
/api/stock-prices?stock=goog&like=true
/api/stock-prices?stock=goog&stock=msft
/api/stock-prices?stock=goog&stock=msft&like=true

Example return:

{"stockData":{"stock":"GOOG","price":"786.90","likes":1}}
{"stockData":[{"stock":"MSFT","price":"62.30","rel_likes":-1},{"stock":"GOOG","price":"786.90","rel_likes":1}]}
